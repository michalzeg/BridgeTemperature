# BridgeTemperature

### Description
Temperature distribution calculator for bridge sections
Main features:
- Developed in C#
- Unrestricted geometry, material and temperature distribution 
- Sections with voids
- Build-in Eurocode temperature distributions
### Built With
* .NET
*  WPF
